# Instrucciones para agregar tus imágenes

Coloca tus imágenes aquí con los siguientes nombres:

## Imágenes requeridas:

1. **profile.jpg** - Tu foto de perfil (aparece en el círculo del header)
   - Tamaño recomendado: 400x400 píxeles (cuadrada)

2. **retrato-1.jpg** - Para la tarjeta "Mirada Urbana"
   - Tamaño recomendado: 800x1000 píxeles (vertical)

3. **blancoynegro-1.jpg** - Para la tarjeta "Sombras"
   - Tamaño recomendado: 800x1000 píxeles (vertical)
   - Idealmente en blanco y negro

4. **paisaje-1.jpg** - Para la tarjeta "Amanecer en la pedriza"
   - Tamaño recomendado: 800x1000 píxeles (vertical)

## Formatos aceptados:
- JPG/JPEG (recomendado)
- PNG
- WebP

## Nota:
Los demás proyectos ya tienen imágenes de Unsplash funcionando. Si quieres cambiar más imágenes, avísame y actualizo el código.
